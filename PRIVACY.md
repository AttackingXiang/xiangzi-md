# Xiangzi MD 隐私说明

最后更新：2026 年 7 月 10 日

Xiangzi MD 是一款本地优先的 Markdown 编辑器。我们重视你的文档与隐私。

## 本地数据

- 使用 Xiangzi MD 无需注册或登录账号。
- 你打开、创建和编辑的文档默认保存在你选择的本地位置。
- Xiangzi MD 不会主动上传、收集或分析你的文档内容。
- Xiangzi MD 不包含广告、用户行为分析或使用情况追踪功能。

## 网络访问

以下功能可能产生网络请求：

- 检查软件更新时，应用会访问 GitHub；访问失败时可能回退到 Gitee。
- 文档包含远程图片且你允许加载时，应用会向图片所在服务器请求该图片。图片服务器可能按照其自身政策记录 IP 地址、请求时间等常规网络信息。
- 你主动打开使用指南、更新日志、问题反馈等外部链接时，系统浏览器会访问对应网站，并受该网站隐私政策约束。
- “支持项目”中的支付宝、微信和 PayPal 二维码均随应用保存在本地，打开支持页面本身不会自动发起付款或上传信息。你主动扫码或打开 PayPal 后，付款将由相应支付平台处理，并受该平台的服务条款和隐私政策约束。收款方可能根据支付平台提供的信息看到付款人的昵称、部分账户信息、金额和付款备注。

## 第三方工具

当你启用 Word 导入或导出等相关功能时，Xiangzi MD 可能调用你自行安装的 Pandoc。相关处理在你的设备上完成。

## 问题反馈

如果你通过 GitHub Issues 主动提交问题，请不要附带私人文档、密码、访问令牌或其他敏感信息。你提交的信息将受 GitHub 的服务条款和隐私政策约束。

如对本说明有疑问，请通过项目的 [GitHub Issues](https://github.com/AttackingXiang/xiangzi-md/issues) 联系我们。
