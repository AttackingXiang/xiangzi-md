# 欢迎使用 Xiangzi MD

这是一个**所见即所得**的 Markdown 编辑器，体验类似 Typora。

## 功能

- 打开单个 `.md` 文件
- 打开整个文件夹，在侧边栏浏览
- 多标签页编辑
- 所见即所得 / 源码模式切换
